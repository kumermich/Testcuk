InheritanceDocumented.circle = Circle(5)

A Circle object representing a geometric circle.

This variable holds an instance of the "Circle" class, which represents a circle. It is created with a radius of 5. The "circle" object allows you to calculate and access the area of the circle using the overridden "area" method.

Note
The "Circle" class is a derived class from the base class "Shape" and provides its own implementation of the "area" method.
