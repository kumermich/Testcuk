This code file is created in order to showcase the concept of private variables in Python.

Description

This file contains a simple class "Person" representing a person with private attributes and methods.

Note
In Python, private attributes and methods are denoted with double underscores (__). However, Python does not provide true data encapsulation; these attributes and methods can still be accessed if needed.
Warning
Directly accessing private attributes and methods should be avoided as it goes against the principles of encapsulation.
