Detailed Description

A class representing a rectangle. 

This class inherits from the base class "Shape" and provides an implementation of the "area" method specifically for
calculating the area of a rectangle.

@note The "Rectangle" class overrides the "area" method to calculate the area of a rectangle using the formula: width * height.
