Description

This script illustrates the concept of properties, getters, setters, and deleters in Python classes. The class "Customer" is used to represent customers with their first names and last names. Properties are defined for email addresses and full names. Additionally, setter and deleter methods are provided for manipulating the full name attribute.

Note
The use of properties, getters, setters, and deleters in Python classes provides an elegant way to control attribute access and encapsulate data while allowing for customized behavior when accessing, setting, or deleting attributes.
