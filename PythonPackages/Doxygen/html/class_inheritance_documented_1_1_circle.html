Detailed Description

A class representing a circle. 

This class inherits from the base class "Shape" and provides an implementation of the "area" method specifically for
calculating the area of a circle.

@note The "Circle" class overrides the "area" method to calculate the area of a circle using the formula: pi * r^2,
where "r" is the radius of the circle.
