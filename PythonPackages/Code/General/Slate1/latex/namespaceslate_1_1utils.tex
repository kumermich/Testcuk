slate.\+utils.\+normalise\+\_\+whitespace (\begin{DoxyParamCaption}\item[{}]{s}{}\end{DoxyParamCaption})}

\begin{DoxyVerb}Returns a string that has at most one whitespace
character between non-whitespace characters. We
leave a few extra spaces because most NLP parsers
don't tend to care.

>>> normalise_whitespace(' hi   there')
' hi there'
>>> normalise_whitespace('meh\n\n\f')
'meh '
\end{DoxyVerb}
